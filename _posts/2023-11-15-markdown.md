## 怎么让MD更美观

#### 超链接 
###### 目录 
+ doctoc
> npm install -g doctoc
> doctoc yourtargetfile.md
+ [toc]
> 直接在 yourtargetfile.md 文件最前面输入`[toc]`即可
> 缺点：只能生成文字目录，不支持超链接
#### 玩转字体 
###### 字体颜色
+ 借助 html 
```html
<font color=red>我是红色</font>
<font face="黑体" color=green size=5>我是黑体，绿色，尺寸为5</font>
```
> <font color=red>我是红色</font>
> <font face="黑体" color=green size=5>我是黑体，绿色，尺寸为5</font>

`<table><tr><td bgcolor=yellow>给字体加背景色</td></tr></table>
`
> <table><tr><td bgcolor=yellow>给字体加背景色</td></tr></table>
