import requests as req
from bs4 import BeautifulSoup
import base64
s = req.Session()
r = s.get("http://vps1.blue-whale.me:23331/captcha/")
html = r.text
bf = BeautifulSoup(html,"html.parser")
#提取img中编码段
texts = bf.find('img').get('src').split(',')[1]
#解码base64
htmll = base64.b64decode(texts)
#解码后的byte初始化BS
te = BeautifulSoup(htmll,"html.parser")
text = te.find_all('text')
answer = ''
#根据坐标sort
str = sorted(text, key=lambda x: float(x.get('x')),reverse=False)
for i in str:
    answer += i.string
r = s.post('http://vps1.blue-whale.me:23331/captcha/', params={'code': answer})
print(r.text)

分析可知对于 name,password,test须满足的要求是

  • md5(test) ==0
    在PHP中，将md5值以0E开头的一律做 == 0处理，由此，只需要test值可得
  • name != password 并且 sha1(name) == sha1(password)
    sha1加密，x的先验概率等于后验概率，也就是说两个不同的明文sha1加密的结果一定不同，在此有函数，利用 false == false，即设计name&password的格式不能做sha1即可

basic PHP2

题目PHP代码如下： 

if(is_int(stripos($content, 'php')) || is_int(stripos($content, '<'))) {
        echo 'Invalid input';
    } else {
        file_put_contents($filename, $content);
        echo 'Success';
    }

stripos() 函数查找字符串在另一字符串中第一次出现的位置（不区分大小写）。
语法: stripos(string,find,start)
返回值：返回字符串在另一字符串中第一次出现的位置，如果没有找到字符串则返回 FALSE。。

is_int() 函数用于检测变量是否是整数。
