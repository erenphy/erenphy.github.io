函数逻辑为判断(array[i]-num1)array[i] 是否等于num2 || (array[i+2]-num1)array[i+2] 是否等于num3 将判断的结果赋值为v0
每次的sub 函数都会将dword_602064与v0进行或运算
这样要使得最后的dword_602064值为0，需要确保每次vo均为0，即
(byte_602080 - 147) * byte_602080 == -4590 && (byte_602082 - 147)*byte_602082 == -4850
另外
涉及的byte_602080—60208f 都属于.bss段

BSS段（bss segment）通常是指用来存放程序中未初始化的全局变量和静态变量的一块内存区域。BSS是英文Block Started by Symbol的简称。BSS段属于静态内存分配。

整理思路，共有14个sub函数，每个函数内部要求两个未知数（比如x1,x3）各自满足一个一元二次方程。这样整体x2–x13都各自需要满足两个一元二次方程，x2–x13都有唯一解 （x0,x1,x14,x15只需要满足一个一元二次方程）
脚本如下：

import math

a = []
b = [147,999,1414,214,1111,1010,741,114,911,2333,6666,917,1020,54321]

for i in range(0,14):
     deta1 = int(b[i] * b[i] - 4 * c1[i])
     t1 = int((-int(b[i]) + math.sqrt(deta1))/2)

     if t1 == t3 or t1 ==t4 :

     else :
